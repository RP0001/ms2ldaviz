Peaklist file:

The required file format for the MS1 peak list is as follows: a .csv file with header line: ("...,mass,RT,samplename_1, samplename_2,..."), delimiter: '.

The *mass* column must have the name mass or mz (case-insensitive). Retention time is assumed to be the column immediately after mass, and should be in seconds. Any columns before *mass* (e.g. containing peak IDs from other software) will be ignored.

All columns after retention time will be interpreted as MS1 intensities, one column per sample. Missing data can be stored as empty (e.g. 123.45,,567.89), or NA (e.g. 123.45,NA,567.89) in the .csv file. Any intensities that are negative values, cannot be interpreted as a float, or are empty will be ignored and not be stored for further analysis.
